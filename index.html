Publicar na Web (GitHub Pages + Domínio Personalizado)

1 Hospedagem Grátis no GitHub Pages

  1. Crie uma conta gratuita no GitHub (se não possuir uma).
  2. Crie um novo repositório chamado atelie-aquarela (público ou privado).
  3. Suba o arquivo index.html da sua máquina para a raiz desse repositório.
  4. No GitHub, entre nas configurações do repositório (aba Settings) e vá na opção lateral Pages.
  5. Em Build and deployment, defina a Source como Deploy from a branch, selecione a branch main (ou master) e clique em Save.
  6. Em instantes, o seu site estará no ar gratuitamente sob o endereço do GitHub (ex: https://seu-usuario.github.io/atelie-aquarela/).

2 Configurando Domínio Próprio

  1. No painel da provedora onde você comprou seu domínio próprio (ex: Registro.br, HostGator, Cloudflare, Godaddy), acesse as Configurações DNS.
  2. Adicione uma entrada do tipo CNAME:
    • Nome/Host: www
    • Destino/Valor: seu-usuario.github.io (sem o subdiretório do repositório)
  3. Adicione as seguintes quatro entradas tipo A (apontando para os servidores oficiais do GitHub) para que seu domínio funcione sem o "www":
    • 185.199.108.153
    • 185.199.109.153
    • 185.199.110.153
    • 185.199.111.153
  4. Nas configurações de Pages do seu repositório no GitHub, preencha o campo Custom domain com seu domínio próprio (ex: www.seuatelie.com.br) e clique em Save.
